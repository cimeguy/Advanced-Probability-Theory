# 高等概率论知识库 (Advanced Probability Theory)

本 skill 是一份研究生《高等概率论》课程笔记的可检索知识库，建立在测度论基础上，参考教材为严加安《测度与概率》。先修：概率论；最好修过实变函数/实分析。

## 如何使用本 skill

1. **先查索引**：打开 `index.md` —— 一张「概念 → 所在讲 → 一句话定义 → 关键定理」的全局检索表。按用户问题里的术语（中英文/定理号）在其中定位。
2. **再读原文**：索引会指向具体的源笔记文件。源笔记是项目根目录下的 `md文件/高等概率论第N讲.md`（含完整 LaTeX 定义、定理与证明）。需要严格陈述或证明细节时读对应文件。
3. **章节地图**：`chapters/课程地图.md` 给出 8 章与 15 讲的对应关系，用于按主题（而非讲次）检索。

## 检索建议

- 用户问"什么是X / X的定义" → 在 index.md 找概念，给一句话定义，必要时引源文件公式。
- 用户问"X定理怎么证" → 在 index.md 找定理号 → 读对应 `md文件/第N讲.md` 的证明段落。
- 用户问"X和Y什么关系" → 可能跨讲，用 chapters/课程地图.md 定位多个来源。
- 公式用 LaTeX 给出，与源笔记保持一致。
