# 课程地图：8章 ↔ 15讲

教材：严加安《测度与概率》。本课程按测度论框架展开，章与讲不完全一一对应（部分讲跨章）。

```
Ch1 概率空间 ─────────────┬─ 第1讲  集类/σ代数/单调类定理
                          └─ 第2讲(前) 概率测度/扩张定理/L-S测度
Ch2 随机变量与概率分布 ───┬─ 第2讲(后) 随机变量定义/判定/示性函数
                          └─ 第3讲  简单r.v/阶梯逼近/分布/存在性
Ch3 数学期望 ─────────────┬─ 第4讲(后) 期望定义/Levi定理/期望性质
                          └─ 第5讲  积分变换/MCT/Fatou/DCT
Ch4 乘积测度与独立性 ─────┬─ 第4讲(前) 事件与r.v独立性
                          ├─ 第6讲  乘积测度/截口/卷积/转移概率
                          └─ 第7讲  Fubini定理/无穷乘积测度
Ch5 条件期望 ─────────────┬─ 第8讲  符号测度/Hahn分解/Lebesgue分解/R-N
                          ├─ 第9讲  条件期望定义(R-N导数)
                          └─ 第10讲 条件期望性质(塔性质等)
鞅与Markov过程 ───────────── 第11讲 鞅定义/性质/Markov过程
Ch6 收敛性 ───────────────┬─ 第12讲 a.s.收敛/B-C引理/0-1律/依概率收敛
                          ├─ 第13讲 L^r收敛/矩收敛/依分布收敛/弱收敛
                          └─ 第14讲 Helly选择/淡收敛↔弱收敛
Ch7 大数定律 ─────────────┬─ 第14讲(后) 经典LLN
                          └─ 第15讲(前) WLLN判据/Kolmogorov SLLN
Ch8 特征函数与CLT ────────── 第15讲(后) 特征函数/连续性/唯一性/CLT/Lindeberg-Feller
```

## 主题导航

- **想看"概率的公理化基础"** → Ch1（第1-2讲）
- **想看"随机变量、分布、期望、积分"** → Ch2-Ch3（第2-5讲）
- **想看"独立性、Fubini、卷积"** → Ch4（第4、6、7讲）
- **想看"条件期望、Radon-Nikodym"** → Ch5（第8-10讲）
- **想看"鞅"** → 第11讲
- **想看"各种收敛及其关系"** → Ch6（第12-14讲）
- **想看"大数定律(强/弱)"** → Ch7（第14-15讲）
- **想看"特征函数、中心极限定理"** → Ch8（第15讲）
